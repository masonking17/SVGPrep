Scales are functions that map from an input domain to an input range. The values in any dataset are not likely to correspond exactly to pixel measurements for use in data visualizations. Scales provide a convenient way to map those data values to new values useful for visualization purposes. They are typically used to transform (or "map") data values into visual variables (such as position, length and color). Scales are useful because they allow for better visualization of data and make it easier to arrange data in whatever fashion you choose. Scales make it so that data can be arranged in sized accordingly to the computer's pixel measurements.
